# 🔐 Password Generator

A simple Python program that generates strong and random passwords.

## Features

* Generate random passwords
* Uses letters, numbers, and special characters
* Easy to understand and use

## Requirements

* Python 3

## How to Run

```bash
python generator.py
```

## Example Output

```text
Generated Password:
fT8@qL2!mP9#xZ4A
```
